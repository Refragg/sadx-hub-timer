Install:
  Extract the contents of the zip file to a new folder in mods i.e.:
  
  |  system\
  |  mods\
  |  |  some-mod\
  |  |  some-other-mod\
  |  |  sadx-hub-timer\
  |  |  |  mod.ini
  |  |  |  README.txt
  |  |  |  sadx-hub-timer.dll
  |  |  ...
  |  |  ...
  |  |  SADXModLoader.dll
  |  ...
  |  ...
  |  sonic.exe

Controls (configurable through the SADXModManager):
  Unpaused & Paused:

	  Stop timer button (LB): Stop the currently running timer
	  Reset timer button (RB): Reset the timer to 0 (without stopping it)
	
  Paused:

	  Start (held) + Increase hubs to time button (RB): Hubs to time + 1
	  Start (held) + Increase hubs to time button (LB): Hubs to time - 1
	
	  Modifier button (Y) (held) + Pause timing button (LB): Toggle time pauses
    Modifier button (Y) (held) + Manual start button (RB): Start timer manually
