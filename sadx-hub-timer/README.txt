Install:
  Extract the contents of the zip file to a new folder in mods i.e.:
  
  |  system\
  |  mods\
  |  |  some-mod\
  |  |  some-other-mod\
  |  |  sadx-hub-timer\
  |  |  |  mod.ini
  |  |  |  README.txt
  |  |  |  sadx-hub-timer.dll
  |  |  ...
  |  |  ...
  |  |  SADXModLoader.dll
  |  ...
  |  ...
  |  sonic.exe

Controls:
  Unpaused & Paused:

	  LB : Stop the currently running timer
	  RB : Reset the timer to 0 (doesn't stop it)
	
  Paused:

	  Start (held) + RB : Hubs to time + 1
	  Start (held) + LB : Hubs to time - 1
	
	  Y + LB : Toggle time pauses
